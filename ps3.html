Свой сайт для взлома ps3

инструкция по созданию своего хоста

Делать свою страницу мне лень и это очень муторно, поэтому излагаю инструкцию как создать свой сайт для взлома ps3, если по каким-либо причинам ps3tools у вас не работает.

Хостить сайт можно где угодно, даже на бесплатном GitHub.io.

1. Подготовка файлов (Modern 4.80–4.93)

Начнем с HEN: для начала найдите архивы типа ps3hen_3.5.0_<very>_fw.zip. Внутри будут бинарники, html-страница и PKG. Здесь ps3hen enabler html — главный экспорт, бинарник загружается через html, а pkg предназначен для установки после эксплоита.

Для CFW и патча памяти (только fat и slim до модели 25xx) качайте с сайта ps3xploit файлы path_<very>.html для каждой версии и флеш-патчер для NAND/NOR.

2. Структура файлов на сервере

Структура выглядит следующим образом:

корень_сайта/
├── index.html                 (главная страница-инструкция)
├── hen/
│   ├── 4.93/
│   │   ├── enabler.html       (ps3hen_enabler_usb000.html)
│   │   ├── PS3HEN.BIN
│   │   └── installer.pkg
│   ├── 4.92/
│   ├── 4.91/
│   └── universal_48x/         (для 4.80-4.89)
├── cfw/
│   ├── 493.html
│   ├── 492.html
│   └── 48x.html

    ├── HEN_493.pkg

    └── custom_firmware.pup

Динамическое определение версии через JavaScript

Лучше внедрить скрипт автоматического подбора под User-Agent консоли:

<script>
(function() {
    let ua = navigator.userAgent;
    let version = null;
    let match = ua.match(/Firmware\/([0-9.]+)/);
    if (match && match[1]) version = match[1];
    
    let links = {
        '4.93': './hen/4.93/enabler.html',
        '4.92': './hen/4.92/enabler.html',
        '4.91': './hen/4.91/enabler.html',
        '4.90': './hen/4.90/enabler.html',
        'default': './hen/universal_48x/enabler.html'
    };
    
    if (version && links[version]) {
        document.write('<a href="' + links[version] + '" class="btn">▶ Запустить HEN ' + version + '</a>');
    } else {
        document.write('<a href="' + links.default + '" class="btn">▶ Универсальный HEN (4.80-4.89)</a>');
    }
})();
</script>

5. Legacy-версии (3.00–3.55)

Если же у вас прошивка в диапазоне 3.00-3.55, то, во-первых, welcome to the club buddy, а во-вторых, расклад следующий:

  • 3.00–3.15: PSGroove / Rockbox. Вы получите CFW через специальный USB-девайс. Для сайта нужна прошивка самого девайса.
  • 3.20–3.40: PS3 Jailbreak (например, через калькулятор TI-84). Тоже нужен специальный USB.
  • 3.41: Hermes USB exploit. Для работы сайта нужны hermes_exploit.html и бинарник.
  • 3.55: Классический 3.55 CFW (Kmeaw, Rebug). Для сайта нужен cfw_355.pup. Получаете полный функционал без промежуточных патчей памяти.
  • 3.56–3.60: Практически голая зона. Напрямую ничего не поставить, требуется аппаратный даунгрейд программатором (E3 Flasher или Progskeet с пайкой/клипсой). Проверить минимальный порог фабричной прошивки можно утилитой minverchk.
⚠ ГЛАВНОЕ ПРАВИЛО: Если у вас прошивка от 3.20 до 3.55 — НЕ ОБНОВЛЯЙТЕСЬ на официальные новые версии! Вы можете сразу накатить кастомную 3.55 CFW, а уже из-под неё обновиться до любой современной актуальной CFW (например, Evilnat).
